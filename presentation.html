核心设计理念

01

专业化分工，不是万能助手

每个 Agent 有独立的 SOUL 定义，明确职责边界和专业领域。商务只管渠道，运营只管增长，互不越界。

02

CEO 统筹，并行处理

CEO 理解意图后，同时派发任务给三条线并行执行。传统串行等待变成并行协作，效率提升数倍。

03

知识边界声明，防止幻觉

每个 SOUL 都明确声明"不编造数据"。不确定的信息标注"待确认"，杜绝 AI 编造具体数字和人名。

04

开箱即用，10 分钟部署

已整理成开源项目，一键安装脚本 + 完整配置模板 + 使用教程。同事下载后只需填入 API Key 即可运行。
